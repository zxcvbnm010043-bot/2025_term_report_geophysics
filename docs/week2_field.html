地震折射探勘：臨界折射波解析

深入探討雙層水平地質模型下的地震波傳播行為、走時公式推導與關鍵交叉距離 $X_{cr}$ 的應用。

1. 物理模型基礎

地震折射法建立在 斯乃爾定律 (Snell's Law) 的基礎之上。當下層速度 $V_2$ 大於上層速度 $V_1$ 時，波會在界面產生折射。

關鍵概念：臨界角 (Critical Angle)

當入射角達到臨界角 $i_c$ 時，折射波將沿著界面傳遞，形成所謂的「頭波 (Head Wave)」。

sin(ic) = V1 / V2

臨界折射波走時公式推導

1 路徑與時間分解

總走時 $T$ 是由下行、滑行、上行三段路徑之時間組成：

Trefr = (L1 / V1) + (L2 / V2) + (L3 / V1)

詳細幾何代換

L1 = L3 = h / cos(ic)
L2 = x - 2h·tan(ic)

透過三角函數簡化，最終可得線性方程：

T = (x / V2) + Ti

交叉距離 (Crossover Distance)

物理意義： 在此距離，折射波（頭波）剛好追上直接波。

「當 $x > X_{cr}$ 時，折射波將成為第一抵達波（First Arrival），這是地下高速層存在的重要觀測證據。」

圖 4：交叉點 $X_{cr}$ 與截距時間之關係

交叉距離最終計算式

Xcr = 2h · √[(V₂ + V₁) / (V₂ - V₁)]

地球物理參數反算總結

利用截距時間反算厚度 ($h$)

h = (TiV₁V₂) / [2√(V₂² - V₁²)]

利用交叉距離反算厚度 ($h$)

h = (Xcr/2) · √[(V₂ - V₁) / (V₂ + V₁)]
